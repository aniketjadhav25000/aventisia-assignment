# Aventisia | Junior Developer Practical Assignment

**Live Demo:** https://aventisia-assignment-olive.vercel.app/

### **Project Overview**
This repository contains the front-end implementation of a **Knowledge Base Management Dashboard**, developed as a practical assessment for the Junior Developer position at **Aventisia**. 

The goal was to replicate a high-fidelity Figma design using **React** and **Tailwind CSS**, focusing on a modular component architecture and pixel-perfect UI execution.

---

### **Core Features**

* **Knowledge Base Grid**: A clean, organized layout for managing data entries, housed within a professional light-grey bordered container (`bg-[#F9FAFB]`).
* **Interactive Side Drawer (Modal)**: A fully functional "Create New Knowledge Base" form that slides out from the right, matching the specific design requirements for data input.
* **Case-Sensitive Asset Management**: Fixed logo and image pathing issues to ensure 100% visibility on Linux-based production environments like Vercel.
* **Branded UI Styling**: Used the exact brand palette: **Primary Indigo (#4F46E5)** and **Secondary Navy (#1E1B4B)**.
* **Navigation System**: A categorized sidebar with sections for **MY PROJECTS**, **ORCHESTRATOR**, and **ADMIN**, featuring a clear active-page indicator.
* **Pagination Footer**: A production-ready footer showing row counts, items-per-page selection, and navigation controls.
